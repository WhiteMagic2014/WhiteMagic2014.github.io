初章:MagicSpace的诞生

发表于 2017-03-07 | 分类于 hexo

  因为网上有好多类似的教程了，我也不喜欢在把别人说过的话再说一遍，没啥意义。在此自己总结一下，分为几个部分吧：

域名

  域名的购买是在godaddy上买的，解析也是在那边完成的，有的教程用的是百度，阿里 当然都一样，能解析就行了。

服务器

  这里用的是github来当做服务器，把页面上传到git后，再把自己的域名解析到git就好了。

页面本身

hexo

  有了域名和服务器，以及解析后 随便糊个index就可以算是建好了，不过这么随便的话弄这东西的意义在哪？ 所以得好好写页面啊是不是？等等！我TM是在写博客啊！在之前还要写html吗！是不是有点喧宾夺主？
  恩，所以hexo就为此而生，简单一句话，就是hexo可以把 符合HEXO语法 的代码转为html(？)来省去自己写页面的这个步骤。

theme

  主题其实可有可无，不过选一个自己喜欢的风格难道不是更棒！PS：theme属于hexo层级下的一个类别，依附于hexo。
