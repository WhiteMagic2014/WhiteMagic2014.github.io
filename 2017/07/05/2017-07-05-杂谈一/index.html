杂谈一

发表于 2017-07-05 | 分类于 杂谈

  
  叫杂谈一，其实根本不是杂谈，其实就是我随便写点牢骚话，预防不知道哪天还会有个二出来，为了统一化，就叫这个名字了。
  最近的情况就是毫无干劲，好困，好累，打不起精神。虽然脑海中知道要做什么，可是遇到了毫无头绪的问题，不知道怎么解决，好烦躁。java对于界面的亲和力真的几乎就是。。。哎，看了下cocos2x，unity，一个js，一个c#，首先是真的不能再三心二意下去了，然后这些引擎的界面怎么说，就像ps，并不是我期望的那样。。。哎心好累。
  对于游戏的定义自己还没有明确，要做成什么样子的也没有明确，详细设计还不够。现在就遇到了这样的大问题。不知道要针对哪个平台。。。了解了各种不同的方向后，发现自己缺少的东西太多了，但是现在的情况不能再开新坑去学东西了，先得吧手上的东西吃透。这算是一个打击吧，弄的我真的是什么都不想去管了，一个人也快疯了。找不到有相同想法的人啊，也不知道哪有前辈可以去讨教讨教。总之感觉自己还是有点浪费时间。
  现在的时间2017年07月05日，其实这毕业一年的收获基本也是稀里糊涂的，感觉和之前差不多，对未来有些担忧啊。看来我过的还是太安逸了啊啊啊啊啊啊！！！！
不要觉得你很努力，你在码这篇无聊的东西的时候别人可是在学习啊！！！你永远只是个落后者啊！！！
